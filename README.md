# Interview PetalMD - Pokemons

A simple REST api with CRUD actions to query and display pokemons.

```
Pokemons
  GET /api/pokemons
    list Pokemons

  GET /api/pokemons/1
    show a Pokemon

  POST /
    creates a Pokemon

  PUT /api/pokemons/1
    updates a Pokemon

  PATCH /api/pokemons/1
    updates a Pokemon

  DELETE /api/pokemons/722
    delete a Pokemon
```
